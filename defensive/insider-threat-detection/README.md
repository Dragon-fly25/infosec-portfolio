# Insider Threat Detection

Defensive security tools focused on detecting insider threats in financial services and regulated environments.

## Included Techniques
- **Anomalous Logon Detection** (`Detect-LogonAnomalies.ps1`)
- **USB/Removable Media Exfiltration Detection** (`Detect-USBExfiltration.ps1`)

## Overview
Practical detection engineering using Windows Event Logs and Sysmon. 
All tools tested in isolated SANS SEC504 lab with synthetic data only.
